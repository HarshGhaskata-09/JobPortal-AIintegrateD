PROJECT OVERVIEW
----------------
An AI-powered job recommendation platform connecting job seekers with employers
using machine learning algorithms for intelligent job matching.

SYSTEM REQUIREMENTS
-------------------
- Python 3.8 or higher
- pip (Python package manager)
- 500MB free disk space
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection (for initial setup)

------------

Machine Learning:

Frontend:

INSTALLATION STEPS
------------------

Step 1: Extract Project
- Extract the project folder to your desired location
- Open terminal/command prompt in project directory

Step 2: Create Virtual Environment
Windows:
  python -m venv venv

Linux/Mac:
  python3 -m venv venv

Step 3: Activate Virtual Environment
Windows (PowerShell):
  .\venv\Scripts\Activate.ps1

Windows (CMD):
  .\venv\Scripts\activate.bat

Linux/Mac:
  source venv/bin/activate

Step 4: Install Dependencies
  pip install -r requirements.txt

Step 5: Run Application
  python -m app.run

Step 6: Access Application
- Open browser
- Navigate to: http://localhost:5000
- Application is ready to use!

DATABASE STRUCTURE
------------------

Tables:
- users: User accounts (job seekers & employers)
- jobs: Job postings
- job_applications: Application tracking
- saved_jobs: Bookmarked jobs
- portfolios: User portfolios
- portfolio_skills: Portfolio skills
- work_experiences: Work history
- job_categories: Job categories

Relationships:
- User -> Jobs (one-to-many)
- User -> Applications (one-to-many)
- User -> Saved Jobs (one-to-many)
- User -> Portfolio (one-to-one)
- Job -> Applications (one-to-many)

CASCADE DELETE:
- When employer deletes account, all their jobs are auto-deleted
- When user deletes account, all their data is removed

API ENDPOINTS
-------------

Authentication:
  POST /auth/register          - User registration
  POST /auth/login             - User login
  GET  /auth/logout            - User logout
  GET  /auth/profile           - User profile
  POST /auth/delete-resume     - Delete resume
  POST /auth/delete-profile-pic - Delete profile picture

Employer Authentication:
  POST /employer/register      - Employer registration
  POST /employer/login         - Employer login
  GET  /employer/logout        - Employer logout

Jobs:
  GET  /jobs/browse            - Browse all jobs
  GET  /jobs/<id>              - Job details
  POST /jobs/<id>/apply        - Apply for job
  POST /jobs/<id>/save         - Save job
  GET  /jobs/saved             - View saved jobs
  POST /jobs/post              - Post new job (employer)
  GET  /employer/my-jobs       - View my jobs (employer)
  POST /employer/job/<id>/delete - Delete job (employer)

Admin:
  GET  /admin                  - Admin dashboard
  GET  /admin/users            - Manage users
  POST /admin/user/<id>/toggle-active - Activate/Deactivate user
  POST /admin/user/<id>/delete - Delete user
  GET  /admin/jobs             - Manage jobs
  POST /admin/job/<id>/toggle-active - Activate/Deactivate job
  POST /admin/job/<id>/delete  - Delete job
  GET  /admin/applications     - View all applications

Chatbot:
  GET  /chatbot                - Chatbot page
  POST /chatbot/message        - Send message to chatbot

Portfolio:
  GET  /portfolio/<username>   - View portfolio
  POST /portfolio/edit         - Edit portfolio

TROUBLESHOOTING
---------------

Issue: Port 5000 already in use
Solution: 
  - Kill process using port 5000
  - Or change port in app/run.py

Issue: Module not found
Solution:
  - Activate virtual environment
  - Run: pip install -r requirements.txt

Issue: Database error
Solution:
  - Delete instance/job_recommender.db
  - Restart application (auto-recreates)

Issue: Upload fails
Solution:
  - Check file size (max 16MB)
  - Check file type (PDF, DOC, DOCX for resume)
  - Check file type (PNG, JPG, JPEG for images)

Issue: Can't login
Solution:
  - Check credentials
  - Ensure account is active
  - Use correct login page (job seeker vs employer)
